### 摘要

基於 NFT 的藝術處於新一輪數位所有權的最前沿，它使得創作者能夠保留對其作品的控制權，並為其數位媒體創造和出售所有權。

但數位藝術品只是冰山一角：同樣的技術可以解鎖任何獨特的實體或數位資源，包括產品、服務和活動權限。

我們稱這些 NFT -- 用於訪問和確定區塊鏈外資源的所有權

- 智能蒐藏品：智能蒐藏品的應用範圍從數位遊戲、媒體到金融及數位資產。它們也是 Web 3.0 和 [Metaverse](https://en.wikipedia.org/wiki/Metaverse) 的一個關鍵推動因素。一個永遠在線的共享數位宇宙，人們可以在那里工作、玩耍和閒逛。

透過創建這份文件，我們旨在激勵和鼓勵創造者、社群和企業在這一領域繼續創新。

透過智能蒐藏品，藝術家和組織可以用可追溯和高度安全的方式獲取價值，甚至可以將多種形式的價值到 NFT 本身。

## 1. 介紹

我們創造、儲存和交易價值的方式正在改變。基於 NFT 的藝術品 -- 比如在佳士得拍賣會上售出價值6900 萬美元的 Beeple 系列以及 59 萬美元的 Nyan Cat NFT，以及 290 萬美元傑克多爾西（Jack Dorsey）的第一條推（Tweet）- 處於新一輪數位所有權浪潮的前沿，這將改變企業和個人創造、提供和使用方式。

區塊鏈技術作為獨特的數位鑰匙，也可用於鑄造和交易基於 NFT 的藝術品。

這些 NFT 鑰匙，稱為「智能蒐藏品」，可以解鎖幾乎任何東西 -- 包括服務、體驗和數位或實體的產品。

由於智能蒐藏品儲存在區塊鏈上，並保存在區塊鏈錢包中，它們為創造者和擁有者提供了對其數據和商品控制權。

## 2. 什麼是數位蒐藏品？

我們將智能蒐藏品定義為 NFT，它可以解鎖並訪問區塊鏈外的獨特價值。

簡單來說，智能蒐藏品是一個獨特的程式碼，你在區塊鏈錢包中持有，並可透過代幣來訪問使用這些有價值的東西。作為 NFT，智能蒐藏品的程式碼是在區塊鏈上創建的。這意味著每個智能蒐藏品都是唯一的、安全的，並且容易出售或轉讓給另一個用戶。

獨特的訪問權可以指的是一個實體或數位物品（經驗或服務）。事實上，任何人可能出售的東西都可以被稱為「價值」。此價值之所以獨特是因為在你的區塊鏈錢包裡僅有一個智能蒐藏品，使你能夠獲得別人無法獲得的東西。

當我們說價值被保存在區塊鏈之外時，這意味著你正在獲取智能蒐藏品的元數據中沒有編碼的東西。如數位下載或實體事件。可以透過使用 tokenID（每個 NFT 唯一的識別碼）來分配給特定智能蒐藏品的持有人。

提供這種訪問權只需要唯一的識別碼，並非需要智能蒐藏品本身的所有權，因此任何人或組織都可以將新的訪問權映射到任何智能蒐藏品的價值上。

智能蒐藏品本身（在您的錢包中的文件）是不變的。在創建原始 NFT 後，可以隨時映射新價值以供智能蒐藏品的所有者獨自訪問。

由於智能蒐藏品可以解鎖價值，因此它們可以與實體鑰匙和數位密碼相比較。與這些相比，智能蒐藏品有許多改進和好處：它們更安全並且有更多的用途，每個都可以用來訪問多個蒐藏品價值。
一把實體鑰匙（例如一把房子的鑰匙）可以讓你打開一個特定的鎖。這意味著，你可以訪問一個實體位置（如房子）的價值。由於鑰匙可能被盜或被複製。因此允許訪問多個位置會增加風險。此外，如果價值（房子）被出售，安全只能透過更換鑰匙和鎖來保證，因為沒有辦法知道以前的主人是否複製了鑰匙。擁有鑰匙並不足以來證明價值的所有權，因此第三方驗證是必不可少的。

在大多數情況下，數位密碼允許您訪問一個特定的數位帳戶。該帳戶提供的值是由發行該密碼的組織控制的。密碼可以被重複使用，但這也大幅增加了密碼被盜、被猜或被遺忘的風險。

在某些情況下，帳戶可用於訪問附加值（例如，使用您的 Facebook 登入其他服務）。出售帳戶是不切實際的並且會危及安全（必須在帳戶出售後更改密碼），且通常不會這樣做。

與這些相比，智能蒐藏品有許多改進和好處。每個智能蒐藏品都有可能解鎖多種類型的價值，而且這種價值可以由代幣的創造者和其他人無限制地增加。因為智能蒐藏品是由區塊鏈產生的，所以它幾乎不可能被篡改。

銷售智能蒐藏品既簡單又安全； NFT 標準和市場意味著它們可以在流動性市場中自由買賣。 出售後，訪問權限會自動轉移給新所有者，並且由於訊息儲存在可公開驗證的分類帳（區塊鏈）中，因此無需第三方驗證。

### 2.1 使用個案：售票服務

透過查看現實生活中的使用個案，可以輕鬆解釋這些好處。 

例如，智能蒐藏品可用作實體活動（如音樂會）的門票。 讓我們看看它在智能蒐藏品的整個生命週期中如何運作：

- **創作**：每張獨特的門票都是一個智能蒐藏品，售出後會轉移到購買者的區塊鏈錢包中。然後，透過使用支持 NFC 的設備（如智能手機）證明其門票所有權，該錢包的所有者可以訪問該活動。 由於智能蒐藏品 — 「票」保存在區塊鏈上，幾乎不可能篡改它。如果擁有者選擇在活動開始前轉售門票，則無需借助昂貴的仲介，即可簡單且安全地完成此操作。此外，可以對智能蒐藏品進行程序，以便將部分交易費用支付給活動的創建者，使創建者可從發生的任何轉售中受益。
- **附加價值**：活動結束後，活動組織者透過允許所有者訪問音樂會的獨家視頻來為智能蒐藏品增加價值。它們還允許持票人獲得線上商店的獨家折扣，他們可以在那裡購買與活動相關的商品。 創建者可以在同一個智能蒐藏品上堆疊任意數量的價值。 這不需要一次全部添加； 創作者可以無限期地繼續為智能蒐藏品增加價值。
- 銷售：隨著時間的推移，舉辦音樂會的樂隊名聲鵲起，數位蒐藏品的門票繼續累積價值，因為它可以訪問並作為蒐藏品獲得獨特的獎勵。持有人決定出售它並將其列在流行的 NFT 市場上，使收藏家能夠在拍賣中競標。 在競爭性拍賣後，門票及其相關優惠作為蒐藏品以 1,000 美元的價格出售。 發行智能蒐藏品的樂隊會收到一定比例的銷售額，在本例中為 10%。每次售出智能蒐藏品時，他們將繼續收到相關訊息，使他們能繼續從所提供的價值中受益。

智能蒐藏品使創作者能夠安全地限制對數位或實體物品、功能的訪問。 除了上面的票券和事件範例之外，下面還詳細介紹了一些例子，並在圖 3 中進行了說明。

### 3.1. 遊戲和數位蒐藏品

收集、使用和銷售獨特的數位蒐藏品，使用戶能在一個或多個遊戲中訪問獨特的遊戲玩法。

其中包括在線多人遊戲中的數位收藏卡牌遊戲 (CCG) 有獨特的物品和資源（武器、服裝、物品）。
單個智能蒐藏品可以解鎖一個或多個遊戲的訪問權限。 例如，發行商可以讓一個蒐藏品在多個遊戲中解鎖相同的服裝款式，從而使用戶能夠更好地定義自己的數位身份。

此外，NFT 和智能蒐藏品還可以替換用戶帳戶和密碼。 在這種情況下，出售帳戶就像交易智能蒐藏品一樣簡單：持有智能蒐藏品的人可以訪問該帳戶。

這對發行商來說也是有益的，他們目前在二級市場上出售帳戶時不會受益：智能蒐藏品可以編碼以在每次出售帳戶時以一定比例的價格獎勵遊戲發行商。

### 3.2. 金融

可以為智能蒐藏品分配對股息、證券、財產、企業等的權利或契約。 此外，智能蒐藏品可用於重新分配價值。 例如，可以將智能蒐藏品指定為遺囑中的受益人，使某人可以將其遺產轉讓給另一個人，或者在多個 NFT 所有者的情況下將其轉讓給多個人。

### 3.3. 憑證和身份

智能蒐藏品可用作一種獨特且安全的方法來驗證個人身份或其他憑據，並使用該訊息啟用或禁用對特定資源或操作的訪問。 在這種情況下，智能蒐藏品將無法交易以防止身份欺詐。 潛在用途包括投票、身份證和訪問金融帳戶。 一個致力於此的項目是 IdToken。

### 3.4. 表決

NFT 和智能蒐藏品可以為所有者提供投票權，使他們能夠對平台、業務、社區組織、DAO（見下文）甚至國家的運行方式發表意見。

每個投票代幣（智能蒐藏品）都使所有者能夠進行投票。 根據設置，所有者能夠擁有多個投票代幣或僅擁有一個，並且代幣可能是可交易的或不可交易的。 在某些情況下，用戶可以將他們的投票權分配給另一個人，以便他們代表他們投票（同時仍保留其智能蒐藏品的所有權）。

許多 DAO 目前使用 ERC-20 代幣（加密貨幣）作為投票代幣，使人們可以根據他們持有的代幣數量進行投票，並根據需要進行投票（二次投票）。 可以類似地使用智能蒐藏品，但也可以啟用其他投票選項。

例如，稀有的智能蒐藏品可能比更常見的智能蒐藏品提供更大的投票權重（更多選票）。

### 3.5. 去中心化自治組織 (DAO)

去中心化自治組織（DAO）是一個完全去中心化的組織，它根據創建時程式碼的規則自主運行。 DAO 沒有 CEO 或董事會，而是由其相關代幣的所有者管理，這使他們能夠對組織下一步採取的行動擁有投票權。

### 3.6. 虛擬資源

智能蒐藏品是一種更易於使用、更安全的密碼替代品。 用戶無需為不同的網站使用多個安全密碼，而是使用他們的數位（區塊鏈）錢包登錄。 這個數位（區塊鏈）錢包可以容納一個或多個智能蒐藏品，使用戶能夠訪問他們的線上帳戶。

Sequence 和 Rainbow Wallet 等項目已經使這成為現實。 許多在線用戶使用他們的 Facebook 或 Google 帳戶來達到類似的結果； 然而，依賴這些第三方會引起嚴重的隱私問題。 Smart Collectibles 提供了一種加密安全的替代方案，可以保護用戶的隱私。

此外，在 NFT 市場上出售智能蒐藏品很容易，而且這樣做不會損害您的其他帳戶。 相比之下，不僅很難為 Google 或 Facebook 帳戶定價和出售，而且透過這樣做，您將向買方提供您使用該帳戶所獲得的一切。

### 3.7. 媒體所有權

NFT 藝術只是冰山一角。 智能蒐藏品可用於為藝術、音樂、3D 作品甚至基於文本的作品的所有者提供功能。 所有者將能夠創建他們媒體的虛擬畫廊供朋友查看。

### 3.8. 物質資源

使用支持 NFC 的設備，例如智能手機。智能蒐藏品可以訪問實體資源（並防止未經授權的用戶訪問）。 例如，製造商可以使用智能蒐藏品來允許員工訪問和使用特定的機器。 這可以與憑證相結合，以確保員工只使用他們經過培訓可以使用的設備，或其他功能以防止在工作時間之外使用。

### 3.9. 時尚

智能蒐藏品將使高端時尚零售商和最終用戶能夠透過將每件獨特的作品與智能蒐藏品聯繫起來來驗證真品並防止偽造。 用戶將驗證每件作品、檢查所有權歷史、訪問護理指南等。

實際上，這很難實現，因為可以在不更新代幣記錄的情況下出售實體資產。 一種解決方案是提供集中式身份驗證服務，管理所有權的可轉讓性，在代幣記錄更新之前將產品和費用託管在託管中。

### 3.10. 混合使用

使用智能蒐藏品的最大好處之一是可以在單個智能蒐藏品上實施多個不同的個案。 這為創作者、社區和企業提供了幾乎無限的機會，為他們的追隨者和客戶創造價值。

## 4. 智能蒐藏品如何運作？

智能蒐藏品是一種儲存在區塊鏈上的代幣。 正是底層區塊鏈技術以及構成代幣的程式碼為智能蒐藏品提供了使其與其他獲取價值的方式區別開來的好處。 

讓我們分解一下：區塊鏈是一個分散式分類帳，這意味著交易歷史（誰擁有什麼）儲存在許多節點上。 這與傳統的集中式分類帳不同，傳統的集中式帳本在中央節點上保存著一份資料副本。 因為有很多副本，所以幾乎不可能丟失或篡改數據。 見圖 4。

![](https://i.imgur.com/vAGlvkh.png)

圖 4 中心化與去中心化網路模型

例如，比特幣區塊鏈追蹤比特幣的交易，比特幣是一種可替代的數字加密貨幣，可用作價值儲存以及購買商品和服務。這些加密貨幣不是智能蒐藏品。

最近，以太坊等區塊鏈啟用了稱為智能合約的附加功能，這些功能是儲存在區塊鏈上的程式。智能合約可用於創建可擁有和交易的代幣。

這些代幣是使用規定如何使用它們的既定標準創建的。使用 ERC-20 標準創建的代幣是可替代的，可以用作貨幣，而 ERC-721 標準用於鑄造不可替代的代幣 (NFT)。

ERC-721 標準確保每個 NFT 都具有可轉移性等功能，並且每個 NFT 都是獨一無二的。 NFT 的獨特性支持創建智能蒐藏品：每個 NFT 都有一個唯一的 tokenID，去中心化應用程序 (dApp) 可以將其用作解鎖對任何類型價值的訪問的輸入。較新的標準 ERC-1155 可用於創建可替代品（如貨幣）和不可替代品（如 NFT 和智能蒐藏品）。

### 4.1. 使用個案：數位劍

例如，想像一把獨特的數位劍，使用 ERC-721 標准在以太坊區塊鏈上創建為 NFT。它唯一的 tokenID 意味著一次只有一個用戶可以擁有這把劍。

如果那把劍除了存在之外沒有別的作用，那麼它就是 NFT，而不是智能蒐藏品。當然，這並不意味著它毫無價值：如果藝術家的需求量很大，則可以以高價購買或出售此 NFT，就像實物藝術品一樣。

然而，如果它的代幣 ID 能夠訪問阻止區塊鏈的更多價值，我們現在將 NFT 歸類為智能蒐藏品。

現在，想像一下同樣的數位劍，但是現在當您在數位錢包中持有該代幣時，您可以在流行的在線遊戲中裝備和使用它，解鎖您以前無法訪問的新能力和權力。

為此，您需要使用您的數位錢包登錄，以便遊戲可以檢查您錢包中代幣的代幣 ID。可以對遊戲進行編程以啟用特定代幣 ID 的不同功能。

只要所有者將智能蒐藏品放在錢包中，就可以繼續使用此功能。如果他們將智能蒐藏品轉讓或出售給其他人，則對獨特價值的訪問權將被轉讓，並且不再可供賣家使用。

### 4.2. 智能蒐藏品與 NFT

所有智能蒐藏品都是 NFT，因此，它們具有與 NFT 相同的屬性——它們不可替代，擁有可追溯的所有者，並且可以交易。

唯一的區別是，我們將用於解鎖對區塊鏈之外的更多價值的訪問的 NFT 歸類為智能蒐藏品。

需要注意的是，並非所有 NFT 都是智能蒐藏品，但所有 NFT 都有可能被歸類為智能蒐藏品。

如果在任何時候，NFT 的所有者（或任何其他人）決定將獲得額外獨特價值的訪問權與該 NFT 的所有權聯繫起來，它就可以成為智能蒐藏品。
目前出售的許多 NFT 不提供從區塊鏈獲得獨特價值的途徑——它們不是智能蒐藏品——但未來可能不會如此。 隨著智能蒐藏品的繼續採用，更多的 NFT 創建者和所有者很可能希望增加額外的價值。

### 4.3. 比特幣是智能蒐藏品嗎？

可替代的代幣，例如比特幣，不是智能蒐藏品。 它們持有的價值可以透過將它們交換為商品或服務來實現，或者透過將它們交換為法定貨幣或其他加密貨幣來實現。

然而，可以想像，這些可替代的代幣可以起到類似的作用。 例如，一家公司可能會為高淨值比特幣持有者提供某些優勢，例如使用私人會員休息室或其他福利。 在這種情況下，比特幣作為一種智能蒐藏品，只需持有即可解鎖對價值的訪問。

### 5. 展望未來：智能蒐藏品、Web 3.0 和元宇宙

提供對幾乎任何實體或數位價值訪問的能力意味著智能蒐藏品有許多有趣的個案，但真正令人興奮的是其更大的影響。
智能蒐藏品提供的功能是第三代互聯網 Web 3.0 背後的關鍵驅動力。基於 Web 1.0 和 Web 2.0 奠定的基礎，Web 3.0 旨在提供一個更加開放的互聯網，使用區塊鏈和人工智能等技術讓人們重新控制他們的數據。

智能蒐藏品將透過允許用戶控制他們生成內容的使用方式及使用不依賴第三方（例如 Facebook 或 Google）的去信任系統來銷售或交易這些內容來實現這一點。這還有一個好處，即個人將能夠匿名進行交易，這是 Web 2.0 解決方案無法實現的。

未來，智能蒐藏品提供的特定訪問權限本身將能夠被標記化，使所有者能夠出售、借出或複製他們擁有的智能蒐藏品的一部分，同時保持對基礎資產的控制。執行此操作的技術尚不可用，但預計在未來可用。

此外，NFT 和智能蒐藏品也是 Metaverse 創造的關鍵 —— 一個人們期待已久的永遠線上的共享數位世界，人們可以在這裏工作、娛樂和閒逛。在 Metaverse 中，人們將以他們的化身為代表，需要擁有一些東西——數字衣服、物品，甚至虛擬土地——這就是 NFT 和智能蒐藏品的用武之地。

在一個您可以隨心所欲的數位世界中，您持有的數位物品將成為表達您身份的重要方式。雖然我們還沒有進入 Metaverse，但諸如 Decentraland（一個建立在區塊鏈上的虛擬世界）和 Fortnite（及其事件和 IP 交互）等世界正在朝著這個方向發展。

希望在未來幾年內，許多這些不同的網路世界將合併為一個元界，人們將能夠在其中自由穿梭。

## 6. 結論

智能蒐藏品提供的可能性幾乎是無窮無盡的，但要使這些成為現實，我們需要在三個方面進行改進：

**創新**——企業、社區和創作者必須在 NFT 的使用方面進行創新，積累價值以創造真正的智能蒐藏品。此功能已經可用，但迄今未得到充分利用。由於 NFT 生態系統仍處於早期階段，企業、社區和創作者有大量機會聲稱自己是第一個實現這項技術真正潛力的人。

**參與**——為了鼓勵創新，用戶必須獎勵那些尋求提供更多價值而不僅僅是元數據的項目。尋找透過購買他們的 NFT 來支持那些提供很少價值的人來積累價值並支持他們的創作者、社區和企業。

**互操作性**——為了實現元界的真正潛力，還必須有一個解決方案，在目前允許創建智能蒐藏品的多個平台之間建立互操作性，以便用戶可以在任何地方無縫轉移 NFT，為系統提供重要的流動性。

無論您是用戶、創作者還是企業，這都是了解 NFT 和智能蒐藏品激動人心的時刻。
